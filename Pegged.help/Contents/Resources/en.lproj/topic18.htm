Playing the Game

----------------------------------------------------------------------------------------------------------------------------------------

The object of Pegged is to finish the game with one peg remaining on the board.


Playing Pegged involves:


Selecting a Board Type

Choose Cross, Plus, Fireplace, Up Arrow, Pyramid, Diamond, or Solitaire from the Options menu. Each board presents a different challenging peg configuration.

Starting a New Game

Choose New from the Game menu.

Jumping a Peg

1 Point to the peg.

2 Drag the peg to the desired location.

Note: Only certain jumps are legal. See "Rules of the Game" for more information.
